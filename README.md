# CRUX_AI-ML

# Machine Learning Projects

This repository contains two machine learning projects implemented to demonstrate both traditional machine learning concepts and deep learning techniques.

## Projects Included

### 1. Decision Tree Classifier from Scratch
- Implemented using Python and NumPy.
- Built entirely from scratch using Object-Oriented Programming (OOP).
- Utilizes GiniGain and Gini for feature selection and node splitting.
- Trained and evaluated on the Titanic dataset for binary classification.

### 2. CNN-Based Digit Recognition
- Implemented using Convolutional Neural Networks (CNNs).
- Trained on the MNIST handwritten digit dataset.
- Performs image classification by recognizing handwritten digits (0–9).
- Demonstrates the application of deep learning for computer vision tasks.
- The Catch here is that instead of predicting the digits using classification we have performed regression to predict continuous values and then match    them to the correct output labels to understand how close values the model is capable in predicting
